Use ruby 2.0.0p0 strongly recommend you use the same version. First of all you must have these packages installed in your system: postgres (>= 9.1.8), ImageMagick, redis (>= 2.6). Once you install these packages run `bundle` and wait while all the dependencies get installed, create file config/database.yml as shown in the example(database.yml.sample), then to run `rake db:setup` wait white all migrations completed. Run the whole project invoke `rails s`, then sidekiq as `bundle exec sidekiq`. The results of the work are shown in tmp/letter_opener.
